Discussion/Writing Prompts

These prompts can be used in sets or spread across several days and are for either individual or collective reflection.

  • Abstraction (Lab 4)
    • What is abstraction? (See Computational Thinking Practice 3: Abstracting and Big Idea 2: Abstraction.)
    • How is abstraction important to programming?
    • How is calling blocks within blocks similar to and different from nesting repeats in repeats?
    • Paul, do we still want somthing like this in U1 or shall we move it to U2? --MF
  • Types of Blocks (Lab 5)
    • What kinds of blocks have you learned about so far?
    • How are each kind of block visually distinguished from the others?
    • What is an example of using a reporter within a command block?
    • What is an example of using a predicate within a conditional?
  • Mathematics of Computing (Lab 5)
    • What does the random block do? How does it work?
    • What does the mod block do? How does it work?
    • What are some programs or tasks that would benefit from using the mod block?
    • What is the difference between or and and?
    • What does the not block do?
