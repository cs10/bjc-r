The goal of this tutorial is to make a script that plays a scale!

The variable "tempo" was made for us. We're going to make our own variable now!

  • Click on the variables tab and click "Make a variable"

  • In the box that pops up - type the name of our variable "my note" (don't type the quotes).

  • Now you'll have blocks to use for your variable (just like we had blocks to use for tempo).
  1. Initialize the value of "my note"
  2. Inside the repeat -
    • Use the value of "my note"
    • Change the value of "my note"



  • Make a script that plays every other note from 50-100.
