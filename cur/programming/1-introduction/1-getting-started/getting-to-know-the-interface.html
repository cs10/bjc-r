Getting to Know the snap Interface

We name the sections of the snap window to make it easy to talk about them. Here is a brief introduction to each section. You can always look up more, if you like, in the snap manual for more detail.

Palette

The area called the palette contains the blocks (individual instructions) that you will use to tell the computer what you want it to do. The blocks are organized into eight categories according to the kind of thing each block does.

Scripting Area

You will drag blocks of code to the Scripting Area to create scripts to control the computer. The script shown here has two blocks: "when green flag is clicked" and "say Hello!"

Stage

The Stage is where sprites—little on-screen robots—act out the code you write. This sprite is saying "hello" because that is what its script told it to do when the green flag was clicked.

Sprite Corral

The Sprite Corral houses all of the sprites that you (or someone) created. You are given one sprite to start with; you can create more if you like. When a sprite is selected, the Scripting Area shows what instructions it understands. The picture here shows only one sprite in the corral.

Tool Bar

The Toolbar across the very top of the screen includes buttons for the File menu for saving and loading your projects, the Cloud menu with options related to your snap account, the Settings menu with advanced options, window resizing buttons , and Project Control Buttons (green flag , pause , and stop ). You can click on these various buttons to see a bit more about what they do.
You can also click on the snap logo for more information.
