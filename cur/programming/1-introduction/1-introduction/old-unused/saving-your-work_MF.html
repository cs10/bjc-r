Saving Your Work

In snap, your work is saved online ("to the cloud").

  1. If it's not open already, open snap by visiting http://snap.berkeley.edu/run .
  2. Check if you are logged in by clicking the cloud button in the toolbar: .
    • Log in if you need to.
    • Create an account if you have not done so already.
    3. Any time you want that work back, you will use the same button and select "Open..." to see a list of your files.
  3. To save your work, use the File button in the tool bar, and choose "Save" or "Save as...".
  4. Give your project a name that will help you remember what the project is.
