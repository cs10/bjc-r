Question 2:

Question 3:

Finally, write a function that takes in a string of text and prints the given text with the given level of brightness. Be sure to have your function set the brightness level back to default after changing it.

Putting it all together:

We have provided a text file for you to parse through and apply what you have learned to change the terminal display given conditions. Change each word in the text according to the follwoing conditions.

  1. If the length of the word is greater than or equal to 6, change the color of the text to blue.
  2. If the word contains the letter 't', change the background color to yellow.
  3. If the word ends with the letter 'e', change the background color to green.
  4. If the word is a palindrome, change the brightness of the word to bright.
