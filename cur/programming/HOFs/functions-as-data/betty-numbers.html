We'll call a number a betty number if each of its digits is at least as large as the digit before it. This means that there are no digits that decrease in value as the number is read from left to right (so all single-digit numbers are betty numbers). Write a predicate that determines whether any positive number is a betty number; you may use any techniques you'ver learned in this class: recursion, higher-order functions, iteration, etc.

Some betty numbers: 14789, 45, 47899, 1122334, 3
Some non-betty numbers: 12342, 9876, 152

Next, write a block that returns a list containing all of the betty numbers in a given range.
