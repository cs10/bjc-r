A couple notes on this assignment:

  • An index is the position of an item in a list. When referring to a column's index, it means the position of the column in the Table. The column's name or label is the text that represents the column.
  • When doing queries, it is usually best to keep your original Table (whether created from scratch or imported using a .csv file) in tact. This means that any queries or changes involving the Table should be done with script variables to avoid changing the original Table. You will see examples of this several times during this assignment.
