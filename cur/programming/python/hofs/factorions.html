Exercise 1.1: Factorian

A number is called a factorion if it is equal to the sum of the factorials of its digits. For example, 145 is a factorion since 145 = 1! + 4! + 5! (Remember that n! is the fancy, mathematical way of denoting the factorial of n, which is the product of all of the numbers from 1 to the number n.) Another, relatively simpler example of a factorion is 2, since 2 is equal to 2! (we're not exclaiming there.) For all of the exercises in this lab you must use the python list comprehension that you learned in the last lab. You cannot use explicit recursion or loops (while and for).

We've started you out with a function split_into_digits that takes in a number and returns a list of its digits. 


>>> split_into_digits(145)
[1, 4, 5]
>>> split_into_digits(12)
[1, 2]
