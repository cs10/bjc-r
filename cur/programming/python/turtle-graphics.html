One of the very first things we did in Snap! was move our sprite across and stage and draw shapes. We can do the same thing in Python with Turtle Graphics. First we need to import the Turtle module so that we can use it in terminal. To do this, type import turtle into the terminal and press enter.

Next, let’s familiarize ourselves to some of the commands we can use with the turtle module. If we want to see the location of our turtle (or "sprite") on the stage, we use the position function. Type turtle.position() into terminal. This should return an x and y coordinate.


>>> turtle.position()
(0.00,0.00)
>>>

Now let’s try to move our turtle across the stage. Use the forward and backward functions by typing turtle.forward(25) or turtle.backward(25) into terminal. You can move the turtle by using different values instead of 25. You can even use negative numbers! You can also use the position function that you just learned above to keep track of where your turtle is after you move it. Try it out!


>>> turtle.forward(25)
>>>


>>> turtle.forward(25)
>>> turtle.backward(25)
>>>

You might have noticed that so far, we’ve only moved our turtle back and forth. We can start moving the turtle all across the stage by changing the angles. Use the right and left functions and specify an angle to change the turtle’s direction. Type turtle.right(90) or turtle.left(90) into terminal and then try to move your turtle. Can you try to draw a square? A pentagon?


>>> turtle.right(90)
>>>


>>> turtle.right(90)
>>> turtle.left(90)
>>>

Some other helpful functions that you can use are turtle.clear() which clears the stage, turtle.setx(x) which changes the x position to the x-value specified, turtle.sety(y) which changes the y position to the y-value specified, and turtle.setpos(x, y) which changes both the x and y positions of the turtle at the same time.
