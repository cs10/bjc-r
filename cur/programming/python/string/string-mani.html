In Python, there is something called "string". It's not the kind of string that you might think of. A string in Python is a way of representing text. A string is denoted by "" or ''. Either one will work. Why then, should there be 2 notations? Here's why: "That's a book!" and ' "He said hello" '. The first one is an example of having the literal apostrophe as a string, the second one is an example of having the literal quotation marks as a string.

To declare a string, you just simply do arr = "Hello World!"

Below are some built-in functions in Python for string:

>>> arr
'Hello World!'
>>> print(arr)
Hello World!
>>> "AA" + "AA"
'AAAA'
>>> "AA" * 2
'AAAA'
>>> print(len("Hi"))
2
>>> print(arr.count("r"))
1
>>> print(arr.index("r"))
8
>>> intro = "Hello, My name is Python"
>>> intro.replace("Python", "Snap!")
'Hello, My name is Snap!'
>>> "replbced".replace("b", "a")
'replaced'

Slicing a String

word = "Hello World!"

word[start:end] # items start to end-1
word[start:] # items start to the rest of the string
word[:end] # items from the beginning to end-1
word[:] # a copy of the whole string

Remember! Everything starts from 0 in Python!

>>> print(word[:3])
Hel
>>> print(word[-3:])
ld!
>>> print(word[3:])
lo World!
>>> print(word[:-3])
Hello Wor
>>> print(word[100:101])
(returns a blank string)
