Exercise 4: Combine

A better way to combine the items of a list is by using a for loop that you write yourself. Try writing your own version of combine in Python that takes in a list of numbers OR strings and returns the combined result (hint: think about what the + operator does for strings and for numbers): 


>>> nums = [1, 2, 3, 4, 5]
>>> strings = ["hello ", "my ", "name ", "is ", "someone?"]
>>> combine(nums)
15
>>> combine(strings)
"hello my name is someone?"

You can test this function by using function number 4.
