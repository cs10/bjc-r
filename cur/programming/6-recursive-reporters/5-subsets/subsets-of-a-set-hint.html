Subset hints

  1. How many subsets does the empty set {} have?
  2. So what should subsets report in the base case? This is the hardest part to get right!
  3. How many subsets of {Apple, Orange, Banana} contain Apple? How many don't?
  4. Is there another way to describe "the subsets of {Apple, Orange, Banana} that don't contain Apple"?
  5. In the Snap! file menu, choose "Libraries..." at the bottom, then choose "List Utilities." You'll find the append block in that library helpful.
  6. Now write the subsets block. Ask for help if you get stuck.
