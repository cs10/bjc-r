Identifying Issues

  1. What issues or problems do you see in the world around you? List a few issues (they can be big or small) that you care about. Here are some sentence starters that might help (you don't have to use them):
    • In a perfect world, there wouldn't be...
    • I get upset/mad every time I think about...
    • The world can be so unfair - I care about issues that affect...
    • The world would be better if we could...
  2. What are some ways that technology might be used to address some of these issues?
  3. What are some things technology can't do to address some of these issues?
